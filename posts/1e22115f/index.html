EDM 推导总结

下面我们讲的 EDM 指的是 Karras 等人在论文 Elucidating the Design Space of Diffusion-Based Generative Models 中提出的 diffusion 统一框架。它不是一个全新的扩散模型，而更像是一次“把扩散模型拆开重装”的工作：把 网络参数化、训练噪声分布、损失权重、采样 ODE、采样步长调度、数值求解器 这些设计选择分离开来，然后给出一套更清晰、更稳定、更高效的推荐方案。

EDM 可以理解为：

用 σ\sigmaσ 作为统一噪声坐标，把扩散模型写成“从高噪声到低噪声解 ODE”的问题；网络学习在任意 σ\sigmaσ 下把带噪图像 xxx 还原成干净图像 yyy；采样时根据 x−Dθ(x;σ)x-D_\theta(x;\sigma)x−Dθ​(x;σ) 估计噪声方向，然后沿 ODE 一步步去噪。

EDM 的核心思想

传统 DDPM 通常写成：

q(xt∣x0)=N(xt;αˉtx0,(1−αˉt)I)q(x_t \mid x_0) = \mathcal N \left( x_t; \sqrt{\bar\alpha_t}x_0, (1-\bar\alpha_t)I \right)q(xt​∣x0​)=N(xt​;αˉt​​x0​,(1−αˉt​)I)

也就是：

xt=αˉtx0+1−αˉtϵ,ϵ∼N(0,I)x_t = \sqrt{\bar\alpha_t}x_0 + \sqrt{1-\bar\alpha_t}\epsilon, \qquad \epsilon \sim \mathcal N(0,I)xt​=αˉt​​x0​+1−αˉt​​ϵ,ϵ∼N(0,I)

EDM 换了一个更直观的写法：直接用噪声标准差 σ\sigmaσ 表示噪声强度。

令真实数据为 yyy，加噪后的样本为：

x=y+n,n∼N(0,σ2I)x = y + n, \qquad n \sim \mathcal N(0, \sigma^2 I)x=y+n,n∼N(0,σ2I)

也就是：

x=y+σϵ,ϵ∼N(0,I)x = y + \sigma \epsilon, \qquad \epsilon \sim \mathcal N(0,I)x=y+σϵ,ϵ∼N(0,I)

这里的 σ\sigmaσ 就是噪声等级。 σ\sigmaσ 大，图像几乎全是噪声；σ\sigmaσ 小，图像接近真实数据。

所以 EDM 的基本建模对象是：

p(x;σ)=pdata(y)∗N(0,σ2I)p(x;\sigma) = p_{\text{data}}(y) * \mathcal N(0,\sigma^2 I)p(x;σ)=pdata​(y)∗N(0,σ2I)

也就是“真实数据分布经过高斯模糊后的分布”。

从 score 到 denoiser

扩散模型里有一个非常核心的量，叫 score：

∇xlog⁡p(x;σ)\nabla_x \log p(x;\sigma)∇x​logp(x;σ)

它表示：在噪声等级 σ\sigmaσ 下，当前点 xxx 应该往哪个方向移动，才能更接近高概率区域。

但 EDM 不直接让网络预测 score，而是让网络预测 denoised image，即：

Dθ(x;σ)≈yD_\theta(x;\sigma) \approx yDθ​(x;σ)≈y

也就是：给定带噪图像 x=y+σϵx = y + \sigma\epsilonx=y+σϵ，网络输出干净图像 yyy 的估计。

为什么 denoiser 和 score 是等价的？这里用到 Tweedie 公式：

∇xlog⁡p(x;σ)=E[y∣x]−xσ2\nabla_x \log p(x;\sigma) = \frac{\mathbb E[y \mid x] - x}{\sigma^2}∇x​logp(x;σ)=σ2E[y∣x]−x​

如果我们用神经网络 Dθ(x;σ)D_\theta(x;\sigma)Dθ​(x;σ) 近似条件期望：

Dθ(x;σ)≈E[y∣x]D_\theta(x;\sigma) \approx \mathbb E[y \mid x]Dθ​(x;σ)≈E[y∣x]

那么 score 可以写成：

∇xlog⁡p(x;σ)≈Dθ(x;σ)−xσ2\nabla_x \log p(x;\sigma) \approx \frac{D_\theta(x;\sigma)-x}{\sigma^2}∇x​logp(x;σ)≈σ2Dθ​(x;σ)−x​

反过来也可以写成：

Dθ(x;σ)≈x+σ2∇xlog⁡p(x;σ)D_\theta(x;\sigma) \approx x + \sigma^2 \nabla_x \log p(x;\sigma)Dθ​(x;σ)≈x+σ2∇x​logp(x;σ)

所以：

预测 denoised image，本质上等价于预测 score。

这点非常重要。EDM 之所以喜欢用 denoiser，是因为它在训练和采样时更稳定、更直观。

概率流 ODE 推导

EDM 把采样过程看成一个连续时间 ODE。我们从高噪声开始：

x(σmax⁡)∼N(0,σmax⁡2I)x(\sigma_{\max}) \sim \mathcal N(0,\sigma_{\max}^2 I)x(σmax​)∼N(0,σmax2​I)

然后逐渐降低噪声：

σmax⁡→σmin⁡→0\sigma_{\max} \to \sigma_{\min} \to 0σmax​→σmin​→0

最终得到干净图像。

对于 variance exploding 形式的扩散过程，有概率流 ODE：

dxdt=−σ˙(t)σ(t)∇xlog⁡p(x;σ(t))\frac{dx}{dt} = -\dot\sigma(t)\sigma(t) \nabla_x \log p(x;\sigma(t))dtdx​=−σ˙(t)σ(t)∇x​logp(x;σ(t))

该方程具体推导过程见文末

如果令时间变量直接等于噪声强度：

t=σt = \sigmat=σ

那么：

σ˙(t)=1\dot\sigma(t)=1σ˙(t)=1

于是：

dxdσ=−σ∇xlog⁡p(x;σ)\frac{dx}{d\sigma} = -\sigma \nabla_x \log p(x;\sigma)dσdx​=−σ∇x​logp(x;σ)

代入刚才的 score-denoiser 关系：

∇xlog⁡p(x;σ)≈Dθ(x;σ)−xσ2\nabla_x \log p(x;\sigma) \approx \frac{D_\theta(x;\sigma)-x}{\sigma^2}∇x​logp(x;σ)≈σ2Dθ​(x;σ)−x​

得到：

dxdσ=−σ⋅Dθ(x;σ)−xσ2\frac{dx}{d\sigma} = -\sigma \cdot \frac{D_\theta(x;\sigma)-x}{\sigma^2}dσdx​=−σ⋅σ2Dθ​(x;σ)−x​

化简：

dxdσ=x−Dθ(x;σ)σ\frac{dx}{d\sigma} = \frac{x - D_\theta(x;\sigma)}{\sigma}dσdx​=σx−Dθ​(x;σ)​

这就是 EDM 采样 ODE 的核心形式：

dxdσ=x−Dθ(x;σ)σ\boxed{ \frac{dx}{d\sigma} = \frac{x - D_\theta(x;\sigma)}{\sigma} }dσdx​=σx−Dθ​(x;σ)​​

它的含义很漂亮：

  • xxx 是当前带噪图像；
  • Dθ(x;σ)D_\theta(x;\sigma)Dθ​(x;σ) 是网络估计的干净图像；
  • x−Dθ(x;σ)x-D_\theta(x;\sigma)x−Dθ​(x;σ) 是“当前噪声成分”；
  • 除以 σ\sigmaσ 后得到沿噪声尺度变化的速度。

所以采样过程就是沿着这个 ODE，从大 σ\sigmaσ 积分到小 σ\sigmaσ。

EDM 的采样公式

假设当前噪声等级是 σi\sigma_iσi​，下一个噪声等级是 σi+1\sigma_{i+1}σi+1​，其中：

σi>σi+1\sigma_i > \sigma_{i+1}σi​>σi+1​

当前状态为 xix_ixi​。根据 ODE：

dxdσ=di=xi−Dθ(xi;σi)σi\frac{dx}{d\sigma} = d_i = \frac{x_i - D_\theta(x_i;\sigma_i)}{\sigma_i}dσdx​=di​=σi​xi​−Dθ​(xi​;σi​)​

用一阶 Euler 方法：

xi+1=xi+(σi+1−σi)dix_{i+1} = x_i + (\sigma_{i+1}-\sigma_i)d_ixi+1​=xi​+(σi+1​−σi​)di​

即：

xi+1=xi+(σi+1−σi)xi−Dθ(xi;σi)σi\boxed{ x_{i+1} = x_i + (\sigma_{i+1}-\sigma_i) \frac{x_i-D_\theta(x_i;\sigma_i)}{\sigma_i} }xi+1​=xi​+(σi+1​−σi​)σi​xi​−Dθ​(xi​;σi​)​​

因为 σi+1−σi<0\sigma_{i+1}-\sigma_i < 0σi+1​−σi​<0，所以这一步实际上是在“减去噪声方向”。

  • DDPM / DDIM 的习惯（倒计时）：使用的是物理时间步 ttt。定义 t=0t = 0t=0 是干净图像，t=Tt = Tt=T（如 1000）是纯噪声。因此，在采样生成时，时间是倒着走的，从 t=T→T−1→⋯→0t = T \rightarrow T - 1 \rightarrow \cdots \rightarrow 0t=T→T−1→⋯→0。
  • EDM 的习惯（正向计步）：使用的是算法循环的步数计数器 iii。定义 i=0i = 0i=0 是采样的开始（纯噪声），i=Ni = Ni=N（如 30）是采样结束（干净图像）。因此，在采样生成时，计数器是正着走的，从 i=0→1→⋯→Ni = 0 \rightarrow 1 \rightarrow \cdots \rightarrow Ni=0→1→⋯→N。

EDM 更推荐用二阶 Heun 方法。先做 Euler 预测：

x^i+1=xi+(σi+1−σi)di\hat x_{i+1} = x_i + (\sigma_{i+1}-\sigma_i)d_ix^i+1​=xi​+(σi+1​−σi​)di​

然后在新位置计算新的斜率：

di+1=x^i+1−Dθ(x^i+1;σi+1)σi+1d_{i+1} = \frac{ \hat x_{i+1} - D_\theta(\hat x_{i+1};\sigma_{i+1}) } {\sigma_{i+1}}di+1​=σi+1​x^i+1​−Dθ​(x^i+1​;σi+1​)​

最后取两个斜率的平均：

xi+1=xi+(σi+1−σi)di+di+12x_{i+1} = x_i + (\sigma_{i+1}-\sigma_i) \frac{d_i+d_{i+1}}{2}xi+1​=xi​+(σi+1​−σi​)2di​+di+1​​

也就是：

xi+1=xi+(σi+1−σi)12[xi−Dθ(xi;σi)σi+x^i+1−Dθ(x^i+1;σi+1)σi+1]\boxed{ x_{i+1} = x_i + (\sigma_{i+1}-\sigma_i) \frac{1}{2} \left[ \frac{x_i-D_\theta(x_i;\sigma_i)}{\sigma_i} + \frac{\hat x_{i+1}-D_\theta(\hat x_{i+1};\sigma_{i+1})}{\sigma_{i+1}} \right] }xi+1​=xi​+(σi+1​−σi​)21​[σi​xi​−Dθ​(xi​;σi​)​+σi+1​x^i+1​−Dθ​(x^i+1​;σi+1​)​]​

这就是 EDM 采样算法的核心。论文和官方实现也提供了对应代码与采样器。

EDM 的噪声调度

EDM 认为采样时 σ\sigmaσ 的选择非常关键。不是简单线性下降，而是使用 Karras noise schedule：

σi=(σmax⁡1/ρ+iN−1(σmin⁡1/ρ−σmax⁡1/ρ))ρ\sigma_i = \left( \sigma_{\max}^{1/\rho} + \frac{i}{N-1} \left( \sigma_{\min}^{1/\rho} - \sigma_{\max}^{1/\rho} \right) \right)^\rhoσi​=(σmax1/ρ​+N−1i​(σmin1/ρ​−σmax1/ρ​))ρ

其中：

i=0,1,…,N−1i = 0,1,\dots,N-1i=0,1,…,N−1

通常 ρ=7\rho = 7ρ=7。

直觉上，这个 schedule 会在小噪声区域分配更多采样步数。 为什么？因为图像细节主要在低噪声阶段形成。高噪声阶段只决定大致结构，小噪声阶段负责纹理、边缘、颜色、局部细节。

所以 EDM 的采样噪声序列是：

σ0=σmax⁡>σ1>⋯>σN−1=σmin⁡>σN=0\sigma_0 = \sigma_{\max} > \sigma_1 > \dots > \sigma_{N-1} = \sigma_{\min} > \sigma_N = 0σ0​=σmax​>σ1​>⋯>σN−1​=σmin​>σN​=0

最后一步到 σ=0\sigma=0σ=0，得到最终图像。

EDM 的训练目标

EDM 的训练非常直接。

从数据集中采样干净图像：

y∼pdatay \sim p_{\text{data}}y∼pdata​

采样噪声等级：

σ∼ptrain(σ)\sigma \sim p_{\text{train}}(\sigma)σ∼ptrain​(σ)

采样标准高斯噪声：

ϵ∼N(0,I)\epsilon \sim \mathcal N(0,I)ϵ∼N(0,I)

构造带噪输入：

x=y+σϵx = y + \sigma\epsilonx=y+σϵ

然后训练网络：

Dθ(x;σ)≈yD_\theta(x;\sigma) \approx yDθ​(x;σ)≈y

最基本的 denoising loss 是：

L=Ey,ϵ,σ[∣Dθ(y+σϵ;σ)−y∣22]\mathcal L = \mathbb E_{y,\epsilon,\sigma} \left[ |D_\theta(y+\sigma\epsilon;\sigma)-y|_2^2 \right]L=Ey,ϵ,σ​[∣Dθ​(y+σϵ;σ)−y∣22​]

但 EDM 加了一个噪声相关的权重：

L=Ey,ϵ,σ[λ(σ)∣Dθ(y+σϵ;σ)−y∣22]\mathcal L = \mathbb E_{y,\epsilon,\sigma} \left[ \lambda(\sigma) |D_\theta(y+\sigma\epsilon;\sigma)-y|_2^2 \right]L=Ey,ϵ,σ​[λ(σ)∣Dθ​(y+σϵ;σ)−y∣22​]

即：

LEDM=Eσ,y,ϵ[λ(σ)∣Dθ(y+σϵ;σ)−y∣22]\boxed{ \mathcal L_{\text{EDM}} = \mathbb E_{\sigma,y,\epsilon} \left[ \lambda(\sigma) \left| D_\theta(y+\sigma\epsilon;\sigma) - y \right|_2^2 \right] }LEDM​=Eσ,y,ϵ​[λ(σ)∣Dθ​(y+σϵ;σ)−y∣22​]​

这里 λ(σ)\lambda(\sigma)λ(σ) 用来平衡不同噪声等级的训练难度。

一些细节推导与理解

  1. dxdt=−σ˙(t)σ(t)∇xlog⁡p(x;σ(t))\frac{dx}{dt} = -\dot{\sigma}(t)\sigma(t)\nabla_x \log p(x; \sigma(t))dtdx​=−σ˙(t)σ(t)∇x​logp(x;σ(t))

EDM 这里的推导，其实是直接借用了 SDE 框架中关于“方差爆炸（Variance Exploding, VE）”模型的现成结论。

  • 第一步：定义方差爆炸（VE）的前向过程

在连续时间 ttt 下，假设一个前向加噪过程。对于 Variance Exploding (VE) 模型，它的特点是不衰减原图信号，只不断往里面加方差越来越大的噪声。

写成随机微分方程（SDE）的标准形式：

dx=f(x,t)dt+g(t)dwdx = f(x, t)dt + g(t)dwdx=f(x,t)dt+g(t)dw
  • f(x,t)f(x, t)f(x,t) 是漂移项（Drift coefficient），控制信号的衰减趋势。
  • g(t)g(t)g(t) 是扩散项（Diffusion coefficient），控制加入噪声的强度。
  • www 是标准布朗运动（维纳过程）。

因为 VE 模型不缩放原图信号，所以漂移项为 0，即 f(x,t)=0f(x, t) = 0f(x,t)=0。于是它的前向 SDE 形式非常简单：

dx=g(t)dwdx = g(t)dwdx=g(t)dw
  • 第二步：确定扩散项 g(t)g(t)g(t) 到底是什么

我们知道，VE 模型在时刻 ttt 的边缘分布，相当于给原图加了方差为 σ2(t)\sigma^2(t)σ2(t) 的噪声。也就是说，在时间从 0 积分到 ttt 的过程中，累积的方差必须等于 σ2(t)\sigma^2(t)σ2(t)。
